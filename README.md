# E-Commerce Sales & Operations Performance Dashboard

An interactive performance dashboard built in Microsoft Excel to track and optimize business operations, revenue streams, and order fulfillment cycles.

## 📊 Dashboard Preview
![Dashboard Preview](dashboard_preview.png)

## 🔑 Key Features & Technical Skills Shown
- **KPI Metrics Tracking:** Created centralized, formulas-driven summaries for Total Revenue, Total Orders, and Average Order Value (AOV).
- **Data Visualization:** Built dynamic column charts for Category Performance, custom line graphs for Order Fulfillment Trends, and breakdown pie charts for Payment Method Splits.
- **Relational Data Preparation:** Performed operational cleaning on raw transaction records to structure clean pivot back-ends.

## 📈 Key Insights Addressed
- Identified top-performing product categories driving the largest margins of gross revenue.
- Mapped processing bottlenecks within the fulfillment pipeline by segmenting order statuses.
